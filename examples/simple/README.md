# Plugin Python Example

1. Create a virtual environment
   ```shell
   python -m venv venv
   ```
2. Activate the virtual environment
   ```shell
   source venv/bin/activate
   ```
3. Install dependencies
   ```shell
   pip install -r requirements.txt
   ```
4. Run saturn-bot
   ```shell
   saturn-bot --task-file task.yaml
   ```
